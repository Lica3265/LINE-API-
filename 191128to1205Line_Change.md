## <center>更動說明</center>
#### 部分Domain name 的更改
暫時會將這些LIFF functions停止。
這兩個API的小簡介：
1. `liff.scanCode()`  ：讓用戶開啟LINE的QR code 掃描器 並且讀取資料
2. `liff.bluetooth.*` ：搜尋其他藍芽裝置。

##### 影響
| 環境             | 版本                               |
| ---------------- | ---------------------------------- |
| LINE for iOS     | **9.19.0**以後，以上的功能暫時停止 |
| LINE for Android | 現在沒影響，但很快將有其他消息     |
